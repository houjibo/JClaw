## 1. 项目概述

### 1.1 项目定位

**JClaw** = Claude Code 编码能力 + OpenClaw 通用能力 + One Product 意图驱动

- **基础**: JClaw 现有 65,000 行代码（45 工具、8 模型、8 技能）
- **融合**: 选择性融合 OpenClaw + One Product 核心能力
- **目标**: 打造终极 AI 原生研发平台

### 1.2 现有基础（JClaw）

| 能力 | 状态 | 说明 |
|------|------|------|
| **工具系统** | ✅ 45 个工具 | 文件/搜索/Git/MCP/任务/Agent 等 |
| **大模型** | ✅ 8 个模型 | Qwen/Kimi/GLM/MiniMax |
| **技能系统** | ✅ 8 个技能 | 代码审查/单元测试/文档生成等 |
| **MCP 协议** | ✅ 已实现 | 远程工具调用 |
| **REST API** | ✅ 15+ 端点 | 工具/模型/技能 API |
| **WebSocket** | ✅ 已实现 | 实时通信 |
| **终端 UI** | ✅ Picocli+JLine | 交互式 CLI |
| **启动性能** | ✅ <1 秒 | 超快启动 |
| **权限追踪** | ✅ 已实现 | 智能学习用户偏好 |
| **进度追踪** | ✅ 11 种类型 | 工具执行进度 |

### 1.3 融合原则

**优中选优** - 只融合最核心、最有价值的能力

| 优先级 | 标准 | 决策 |
|--------|------|------|
| **P0** | 决定竞争力的关键能力 | 必须融合 |
| **P1** | 提升用户体验和效率 | 重要但可延后 |
| **P2/P3** | 锦上添花 | 根据资源决定 |

## 3. 核心能力筛选

### 3.1 记忆系统（P0）

**OpenClaw 核心能力**:
- MEMORY.md 长期记忆
- 每日日志（memory/YYYY-MM-DD.md）
- 知识萃取

**JClaw 融合方案**:
```
三层记忆架构:
├── L1 工作记忆 (Redis) - 会话级缓存
├── L2 长期记忆 (文件系统) - MEMORY.md + 每日日志
└── L3 知识图谱 (FalkorDB) - 意图图谱 + 代码追溯
```

**新增代码**: ~2,000 行  
**新增文件**: ~15 个

### 3.2 意图驱动（P0）

**One Product 核心能力**:
- 意图识别（自然语言→意图图谱）
- AI 澄清对话（自动生成问题）
- 任务分解（意图→可执行任务）

**JClaw 融合方案**:
```
意图驱动流程:
用户输入 → 意图识别 → AI 澄清 → 任务分解 → Agent 调度 → 执行 → 追溯
```

**复用 JClaw**:
- IntentRecognitionTool（已有工具扩展）
- 大模型 API（已有）
- Task 系统（已有）

**新增代码**: ~4,000 行  
**新增文件**: ~25 个

### 3.3 代码追溯（P0）

**One Product 核心能力**:
- AST 解析（Java/TS）
- 函数级追溯
- 调用链分析

**JClaw 融合方案**:
```
代码追溯系统:
├── AST 解析器 → 提取类/方法/复杂度
├── 代码单元 → 存储到 MySQL
├── 调用链 → 存储到 FalkorDB
└── 3D 可视化 → Three.js 图谱
```

**新增代码**: ~5,000 行  
**新增文件**: ~30 个

### 3.4 影响分析（P0）

**One Product 核心能力**:
- 变更影响范围分析
- 风险评分

**JClaw 融合方案**:
```
影响分析流程:
代码变更 → 检测变更 → 查询调用链 → 分析影响范围 → 风险评分 → 可视化
```

**复用 JClaw**:
- GitTool（已有）
- REST API 框架（已有）

**新增代码**: ~3,000 行  
**新增文件**: ~18 个

### 3.5 多通道（P0）

**OpenClaw 核心能力**:
- 飞书集成
- QQ 集成
- 统一消息抽象

**JClaw 融合方案**:
```
通道架构:
├── ChannelAdapter 接口（统一抽象）
├── FeishuChannelAdapter（飞书实现）
├── QQChannelAdapter（QQ 实现）
└── MessageRouter（消息路由）
```

**复用 JClaw**:
- WebSocket（已有）
- REST API（已有）

**新增代码**: ~3,000 行  
**新增文件**: ~20 个
