**新增文件**:

**新增 API 端点** (10 个):
| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/intent/graph/nodes/intent` | POST | 创建意图节点 |
| `/api/intent/graph/nodes/requirement` | POST | 创建需求节点 |
| `/api/intent/graph/nodes/codeunit` | POST | 创建代码单元节点 |
| `/api/intent/graph/relations` | POST | 创建关系 |
| `/api/intent/graph/query/intent/{id}` | GET | 查询意图链路 |
| `/api/intent/graph/query/requirement/{id}` | GET | 查询需求实现链路 |
| `/api/intent/graph/impact/{id}` | GET | 变更影响分析 |
| `/api/intent/graph/related/{id}` | GET | 查找关联意图 |
| `/api/intent/graph/unimplemented` | GET | 查找未实现意图 |
| `/api/intent/graph/export` | GET | 导出图谱 JSON |

**节点类型**:
- Intent (意图) - 🔴 红色
- Requirement (需求) - 🟢 青色
- CodeUnit (代码单元) - 🔵 蓝色
- Task (任务) - 🟡 黄色

**关系类型**:
- DERIVES (派生)
- IMPLEMENTS (实现)
- RELATES (关联)
- DEPENDS (依赖)
