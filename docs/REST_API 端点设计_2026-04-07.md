## 5. 代码追溯

| 方法 | 端点 | 说明 | 认证 |
|------|------|------|------|
| POST | `/trace/parse` | 解析代码文件 | ✅ |

| POST | `/trace/impact/{id}` | 影响分析 | ✅ |

| GET | `/trace/code-units` | 查询代码单元列表 | ✅ |

      "caller": "UserMapper.findByUsername",
      "callee": "BaseMapper.selectOne",
      "type": "INHERIT",
      "lineNumber": 12

## 🔐 认证说明

**认证方式**: Bearer Token  
**请求头**: `Authorization: Bearer {token}`

**无需认证的端点**:
- `GET /api/health/**` - 健康检查
- `GET /api/tools` - 工具列表（只读）
- `GET /api/tools/{toolName}` - 工具详情（只读）
