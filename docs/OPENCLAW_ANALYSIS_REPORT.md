## 🌐 openclaw-lark 插件功能详解

### 功能特性

| 类别 | 能力 |
|------|------|
| **💬 Messenger** | 读取消息 (群聊/私聊历史、话题回复)、发送消息、回复消息、搜索消息、下载图片/文件 |
| **📄 Docs** | 创建、更新、读取云文档 |
| **📊 Base** | 创建/管理多维表格、表格/字段/记录管理 (CRUD、批量操作、高级筛选)、视图管理 |
| **📈 Sheets** | 创建、编辑、查看电子表格 |
| **📅 Calendar** | 日历和事件管理 (创建/查询/更新/删除/搜索)、参会人管理、忙闲状态检查 |
| **✅ Tasks** | 任务管理 (创建/查询/更新/完成)、任务清单、子任务、评论管理 |

### 高级特性

- **📱 交互式卡片**: 实时状态更新 (思考中/生成中/完成)，敏感操作确认按钮
- **🌊 流式响应**: 消息卡片内实时流式文本
- **🔒 权限策略**: 灵活的 DM 和群聊访问控制策略
- **⚙️ 高级群配置**: 每群设置，包括白名单、技能绑定、自定义系统提示
