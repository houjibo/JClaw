## 📈 使用示例

### 1. AI 意图识别

```java
@Autowired
private AiIntentRecognitionService aiIntentRecognitionService;

// 识别意图
Intent intent = aiIntentRecognitionService.recognizeWithAI("我想创建一个用户管理系统");
System.out.println("意图类型：" + intent.getType());  // feature

// 生成澄清问题
List<String> questions = aiIntentRecognitionService.generateClarificationQuestions(intent);
questions.forEach(System.out::println);
```

### 2. AI 任务分解

```java
@Autowired
private AiTaskDecompositionService aiTaskDecompositionService;

// 分解任务
List<Map<String, Object>> tasks = aiTaskDecompositionService.decomposeWithAI(intent);
tasks.forEach(task -> {
    System.out.println("任务：" + task.get("title"));
    System.out.println("执行者：" + task.get("agent"));
});

// 评估复杂度
int complexity = aiTaskDecompositionService.estimateComplexityWithAI(tasks.get(0));
System.out.println("复杂度：" + complexity + "/10");
```

### 3. AST 解析

```java
@Autowired
private AstParserServiceImpl astParserService;

// 解析单个文件
CodeUnit classUnit = astParserService.parseJavaFile("/src/UserService.java");
System.out.println("类名：" + classUnit.getUnitName());
System.out.println("圈复杂度：" + classUnit.getMetrics().get("complexity"));

// 批量解析目录
int count = astParserService.parseDirectory("/src/main/java/com/example");
System.out.println("解析了 " + count + " 个文件");
```

### 4. 测试推荐

```java
@Autowired
private TestRecommenderService testRecommenderService;

// 推荐测试
List<Map<String, Object>> recs = testRecommenderService.recommendTests("/src/UserService.java");
recs.forEach(rec -> {
    System.out.println("推荐测试：" + rec.get("testName"));
    System.out.println("优先级：" + rec.get("priority"));
});

// 分析覆盖率
Map<String, Object> coverage = testRecommenderService.analyzeCoverage("/src/UserService.java");
System.out.println("行覆盖率：" + coverage.get("lineCoverage") + "%");
```
