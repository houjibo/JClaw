# JClaw 示例项目集合

本目录包含 JClaw 的各种示例项目，帮助你快速上手。

## 示例列表

| 示例 | 难度 | 说明 | 学习时长 |
|------|------|------|---------|
| [hello-world](./hello-world/) | ⭐ | 最简示例，快速开始 | 5 分钟 |
| [spring-boot-demo](./spring-boot-demo/) | ⭐⭐ | Spring Boot 集成 | 15 分钟 |
| [skill-demo](./skill-demo/) | ⭐⭐ | 技能开发示例 | 20 分钟 |
| [mcp-integration](./mcp-integration/) | ⭐⭐⭐ | MCP 协议集成 | 30 分钟 |
| [multi-agent-demo](./multi-agent-demo/) | ⭐⭐⭐⭐ | 多 Agent 协作 | 45 分钟 |

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/houjibo/JClaw.git
cd JClaw/examples
```

### 2. 选择示例

根据你的需求选择合适的示例：

- **新手入门**：从 `hello-world` 开始
- **Spring Boot 开发者**：查看 `spring-boot-demo`
- **技能开发者**：查看 `skill-demo`
- **高级用户**：查看 `mcp-integration` 和 `multi-agent-demo`

### 3. 运行示例

每个示例都有独立的 `README.md`，包含详细的运行说明。

```bash
cd hello-world
cat README.md
```

## 示例详解

### Hello World (⭐)

最简单的 JClaw 使用示例，展示：
- 如何创建 JClaw 客户端
- 如何执行简单指令
- 如何使用工具
- 如何使用技能

**适合人群**：JClaw 新手

### Spring Boot Demo (⭐⭐)

Spring Boot 集成示例，展示：
- 如何添加 JClaw 依赖
- 如何配置 JClaw
- 如何注入和使用 JClawTemplate
- 如何创建 REST API
- 如何异步调用和流式输出

**适合人群**：Spring Boot 开发者

### Skill Demo (⭐⭐)

技能开发示例，展示：
- 技能结构设计
- skill.json 配置
- 技能实现
- 测试和发布流程

**适合人群**：想开发自定义技能的开发者

### MCP Integration (⭐⭐⭐)

MCP 协议集成示例，展示：
- MCP Server 配置
- MCP 资源访问
- MCP 工具调用
- 自定义 MCP Server

**适合人群**：需要集成外部工具的开发者

### Multi-Agent Demo (⭐⭐⭐⭐)

多 Agent 协作示例，展示：
- 协调器配置
- Agent 角色定义
- 协作模式选择
- 人类评审集成

**适合人群**：需要复杂协作场景的开发者
