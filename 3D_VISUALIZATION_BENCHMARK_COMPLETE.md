## 📝 使用方法

### 运行基准测试

```bash
cd ~/.openclaw/workspace/projects/code/core/JClaw

# 运行完整基准测试
mvn test -Dtest=JClawBenchmarkTest

# 运行简化性能测试
mvn test -Dtest=SimplePerformanceTest

# 运行所有测试
mvn test
```

### 查看 3D 可视化

1. 启动前端开发服务器：
```bash
cd src/main/resources/frontend
npm run dev -- --port 3000
```

2. 访问页面：
- 3D 可视化：http://localhost:3000/call-chain-3d

3. 加载示例数据：
- 点击"加载示例数据"按钮
- 查看 3D 代码调用链图谱
