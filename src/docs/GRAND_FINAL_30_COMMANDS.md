## 🚀 使用示例

### 快速开始
```bash
# 查看帮助
help

# 查看版本
version

# 登录
login user@example.com password

# 系统状态
status
```

### Git 工作流
```bash
# 查看状态
git status

# 创建提交
git commit -m "修复 bug"

# 查看分支
branch -a

# 代码审查
review pr 123

# 查看差异
diff HEAD~1 HEAD
```

### 系统管理
```bash
# 查看配置
config

# 切换模型
model qwen3.5-plus

# 查看统计
stats tools

# 健康检查
doctor

# 复制内容
copy Hello World
```

### 会话管理
```bash
# 列出会话
session list

# 恢复会话
resume session-001

# 重命名会话
rename session-001 我的会话

# 清除上下文
clear
```

### 项目管理
```bash
# 初始化项目
init project myapp

# 查看任务

# 查看成本

# IDE 集成
ide install vscode
```

### 1. 子命令模式
- 30 个类实现 100+ 个命令功能


