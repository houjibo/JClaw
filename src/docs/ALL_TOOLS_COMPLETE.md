## 📋 使用示例

### 代码开发流程
```bash
# 1. 初始化项目
init name="my-app" template="java-spring"

# 2. 代码审查
review path="src/Main.java"

# 3. 代码解释
explain path="src/Service.java"

# 4. 代码优化
optimize path="src/Main.java" type="performance"

# 5. 安全扫描
security path="src/Main.java" deep=true

# 6. 运行测试
test path="src/test/" coverage=true

# 7. 构建项目
build clean=true

# 8. 部署
deploy target="production" env="prod"
```

### 任务管理流程
```bash
# 1. 创建任务
task_create title="开发新功能" priority="high"

# 2. 查看任务
task_list status="pending"

# 3. 更新状态
task_update task_id="task_1" status="running"

# 4. 获取输出
task_output task_id="task_1"

# 5. 完成任务
task_update task_id="task_1" status="completed" output="完成"
```

### 多 Agent 协作
```bash
# 1. 创建 Agent
agent_create name="代码审查员" role="reviewer"
agent_create name="开发者" role="coder"

# 2. 查看 Agent
agent_list

# 3. 发送消息
agent_message agent_id="agent_1" message="请审查代码"

# 4. 远程触发
remote_trigger agent_id="agent_1" action="analyze"
```
